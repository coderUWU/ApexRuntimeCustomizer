## ✨ Features

Apex Runtime Customizer provides a comprehensive suite of tools to personalize and enhance your GTA V gameplay, all accessible through an intuitive in-game overlay.

### 🎮 Player Customization
*   **Enhanced Resilience:** Toggle advanced protective states for your character.
*   **Optimized Ammunition Management:** Streamline your weapon usage without constant reloads.
*   **Dynamic Movement:** Experience enhanced character mobility for exploration and interaction.
*   **Situational Presence Module:** Adjust your visibility within the game world.

### 🚗 Vehicle Management
*   **Instant Vehicle Provision:** Access and deploy a wide range of vehicles on demand.
*   **Enhanced Vehicle Durability:** Reinforce your vehicles against damage.
*   **Advanced Customization:** Apply extensive visual and performance modifications to vehicles.
*   **Flight Capabilities:** Grant aerial abilities to traditionally ground-based vehicles.

### 🌍 World Interaction & Navigation
*   **Precision Teleportation:** Instantly relocate to any marked destination or custom saved points.
*   **Environmental Control:** Manipulate in-game weather patterns and time cycles.
*   **Prop Deployment:** Introduce custom objects and elements into the game environment.

### ⚙️ General Utilities
*   **Weapon Parameter Adjustment:** Fine-tune weapon attributes like fire rate and accuracy.
*   **Configurable User Interface:** Personalize the overlay's appearance and layout.

## 📦 Installation

For the most convenient setup, download our pre-compiled `.msi` installer package. This handles all necessary dependencies and ensures a smooth installation process.

1.  Visit the [Releases page](https://github.com/YourName/ApexRuntimeCustomizer/releases).
2.  Download the latest `ApexRuntimeCustomizer-Installer.msi`.
3.  Execute the `.msi` file and follow the provided installation prompts.

## 🕹️ Usage

1.  Launch Grand Theft Auto V and ensure you are loaded into a game session (Story Mode or Online).
2.  Confirm your game's display mode is set to **Borderless Window** or **Windowed**. Fullscreen Exclusive is not supported by overlay applications.
3.  Run `ApexRuntimeCustomizer.exe` (installed via the `.msi` package) with **Administrator privileges** for proper system integration.
4.  Press the `INSERT` key in-game to toggle the customization overlay's graphical interface.
5.  Utilize your keyboard's arrow keys or mouse to navigate and select various enhancement options.
6.  Press `INSERT` again to dismiss the overlay.

## 🛠️ Building from Source

If you prefer to compile Apex Runtime Customizer from its source code, follow these instructions:

*   **Visual Studio 2022:** Ensure the "Desktop development with C++" workload is installed.
*   **Windows SDK:** The latest version is recommended (typically included with Visual Studio installations).
*   **DirectX SDK:** (Optional, for specific legacy DirectX components or deeper integration).

### Compilation Steps
1.  **Clone the repository (recursively for submodules):**
    ```bash
    git clone --recursive https://github.com/YourName/ApexRuntimeCustomizer.git
    cd ApexRuntimeCustomizer
    ```
2.  **Open Solution:** Open `ApexRuntimeCustomizer.sln` in Visual Studio 2022.
3.  **Configure Build:** Set the solution configuration to `Release | x64`.
4.  **Build Project:** Right-click on the `ApexRuntimeCustomizer` project in the Solution Explorer and select "Build".
5.  The compiled executable will be located in the `build/Release/` directory.
